Integración con Culqi

En este tutorial usted va aprender a realizar la integración con Culqi. Si necesita ayuda, contactese con uno de nuestros desarrolladores para asesorarlo.

Usted tiene el token de datos de la tarjeta de crédito de su usuario, ¿ahora qué? Ahora usted les cobra dinero. Esto sucede en su servidor, y la forma más rápida de hacerlo es mediante el uso de una de nuestras bibliotecas de cliente. Si no lo ha hecho, instale su lenguaje favorito ahora. En este tutorial, vamos a mostrar código para PHP, y Java.

  • php
  • java

PHP

En este tipo de integración el formulario de pago aparece como una pequeña ventana modal dentro de tu website como se muestra en la siguiente figura.


Culqi::$LLAVE_SECRETA = "....";
Culqi::$CODIGO_COMERCIO = "....";

echo "<br/>";
echo Checkout::createOrderUrl(array(
"curency" => "PEN",
"amount" => 50,
));

La url obtenida debe ser incrustada en tu formulario de pago de la siguiente forma:


<script src="<?php echo $url ?>"></script>
